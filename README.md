# Excel-Capstone-Project-AI-Powered-Healthcare-Data-Insights-Dashboarding
Cleaned and organized patient health records using Excel for accurate data preparation.
Created an interactive Excel dashboard to visualize patient admissions, health checkups, and critical health trends. 
Generated meaningful insights to support better healthcare decisions and operational efficiency. 
Tools Used: Excel, VLOOKUP, XLOOKUP ,Pivot Tables
Applied statistical formulas and Pivot Tables to identify high-risk patients and monitor department performance.
